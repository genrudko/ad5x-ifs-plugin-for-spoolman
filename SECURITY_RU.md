# Политика безопасности

[English](SECURITY.md)

## Поддерживаемые версии

На стадии до 1.0 поддерживается только последняя beta-версия.

## Сетевой доступ

По умолчанию плагин слушает `0.0.0.0:7913`, а текущая beta-версия API не имеет аутентификации. Используйте его только в доверенной локальной сети. Не публикуйте порт `7913` напрямую в Интернет.

Плагин отправляет рабочие запросы локальному Moonraker и может изменять активную катушку Spoolman. Ограничьте доступ по SSH и доступ к сети принтера.

## Сообщение об уязвимости

Не публикуйте в открытом issue адреса, конфигурацию и другие чувствительные сведения о принтере или сети. Сначала свяжитесь с владельцем репозитория приватно через профиль GitHub.
