# Настройка

[English](configuration.md)

Рабочая конфигурация хранится в:

```text
/usr/data/config/mod_data/ifs_spoolman/config.json
```

Назначения каналов находятся в `assignments.json` и обычно изменяются через веб-интерфейс.

## Параметры

| Ключ | Значение по умолчанию | Назначение |
|---|---:|---|
| `schema_version` | `1` | Версия схемы конфигурации. |
| `moonraker_url` | `http://127.0.0.1:7125` | Локальный адрес API Moonraker. |
| `listen_host` | `0.0.0.0` | Адрес прослушивания HTTP-сервера плагина. |
| `listen_port` | `7913` | HTTP-порт плагина. |
| `slot_count` | `4` | Количество каналов IFS. |
| `poll_interval` | `1.0` | Интервал опроса датчика, секунд. |
| `http_timeout` | `5.0` | Тайм-аут запросов к Moonraker. |
| `spoolman_proxy_timeout` | `10.0` | Тайм-аут проксируемых запросов Spoolman. |
| `switch_confirmation_reads` | `3` | Число совпадающих чтений для подтверждения канала. |
| `switch_confirmation_interval` | `0.25` | Пауза между подтверждающими чтениями. |
| `switch_cooldown` | `2.0` | Минимальная пауза между реальными переключениями. |
| `sync_retry_count` | `3` | Число попыток синхронизации. |
| `sync_retry_delay` | `1.0` | Пауза между попытками. |
| `event_log_max_bytes` | `524288` | Максимальный размер журнала событий до ротации. |
| `event_log_backup_count` | `3` | Количество сохраняемых архивных журналов. |
| `fluidd_integration` | `true` | Параметр интеграции с Fluidd. |

Неизвестные параметры и значения неверного типа отклоняются. После ручного редактирования конфигурации перезапустите только плагин.
