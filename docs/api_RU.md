# HTTP API

[English](api.md)

Базовый адрес по умолчанию:

```text
http://IP_ПРИНТЕРА:7913
```

Текущая beta-версия API предназначена для доверенной локальной сети и не имеет аутентификации.

## `GET /`

Отдельный веб-интерфейс управления.

## `GET /api/status`

Возвращает текущий, необработанный и подтверждённый канал IFS, активную катушку Moonraker, состояние связи, результат синхронизации, счётчики и назначения каналов.

## `GET /api/health`

Возвращает общий статус, время работы и отдельное состояние компонентов IFS sensor, Moonraker и Spoolman.

Возможные значения общего статуса:

- `ok` — все основные компоненты исправны;
- `degraded` — сервис работает, но часть интеграции недоступна;
- `error` — нарушена критически важная часть работы.

## `GET /api/config`

Возвращает действующую публичную конфигурацию и версию приложения.

## `GET /api/spools`

Возвращает список катушек Spoolman через интеграцию Moonraker/Spoolman.

## `POST /api/assign`

Назначает катушку Spoolman определённому каналу IFS. Используется веб-интерфейсом.

## `POST /api/sync`

Принудительно запускает синхронизацию для текущего канала.

## Совместимость

API находится в beta-стадии и может измениться до версии 1.0. Клиентам следует корректно обрабатывать появление дополнительных полей JSON.
