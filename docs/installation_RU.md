# Установка и обновление

[English](installation.md)

## Перед установкой

Убедитесь, что Z-Mod, Moonraker и Fluidd уже работают, а Moonraker видит сервер Spoolman. Установщик плагина не устанавливает и не настраивает сам Spoolman.

## Установка из клона или распакованного репозитория

1. Передайте каталог репозитория на принтер.
2. Подключитесь по SSH как `root`.
3. Перейдите в каталог репозитория.
4. Выполните:

```sh
chmod +x install.sh update.sh scripts/*.sh
./install.sh
```

Для копирования файлов без запуска сервиса:

```sh
./install.sh --no-start
```

Отсутствующие `config.json` и `assignments.json` создаются из примеров. Уже существующие пользовательские файлы не перезаписываются.

## Обновление

```sh
./update.sh --dry-run
./update.sh
```

Процесс обновления:

1. проверяет структуру исходного пакета;
2. создаёт резервную копию рабочих файлов и пользовательских настроек;
3. останавливает только процесс плагина;
4. копирует новую версию;
5. запускает плагин;
6. проверяет `/api/health`;
7. автоматически откатывается при ошибке.

Klipper, Moonraker и MCU принтера этими скриптами не перезапускаются.

## Управление сервисом

```sh
/usr/data/config/mod_data/ifs_spoolman/start.sh
/usr/data/config/mod_data/ifs_spoolman/stop.sh
/usr/data/config/mod_data/ifs_spoolman/status.sh
```

## Удаление

```sh
/usr/data/config/mod_data/ifs_spoolman/uninstall.sh --yes
```

Плагин и интеграция Fluidd удаляются, а конфигурация и журналы копируются в каталог с временной меткой внутри `/usr/data/config/mod_data/`.

Для полного удаления пользовательских данных добавьте `--purge`.
